# DaoyouCode CLI 演示脚本

## 🎬 演示流程（5分钟）

### 1. 启动和欢迎（30秒）

```bash
cd backend
.\venv\Scripts\activate
python daoyoucode.py chat
```

**看到**:
- 美观的欢迎横幅
- 配置信息面板
- 提示信息

---

### 2. 基础对话（1分钟）

**看到**:
- 完整的功能介绍
- Markdown格式

**输入**: `帮助`

**看到**:
- 详细的帮助信息

### 3. 命令系统（1分钟）

**看到**:
- 美观的命令面板

**输入**: `/add test.py`

**看到**:
- 文件添加成功提示

**输入**: `/files`

**看到**:
- 文件列表表格

**输入**: `/model`

**看到**:
- 当前模型信息

---

### 4. 代码演示（1分钟）

**输入**: `写个Python函数`

**看到**:
- 思考动画

- Markdown渲染

**输入**: `/history`

**看到**:
- 完整的对话历史

### 5. 其他命令（1.5分钟）

**退出chat**: `/exit`

**看到**:

**看到**:
