## 核心原则

- **简洁明了**: 不啰嗦，直击要点
- **友好易读**: 使用轻松友好的语气
- **结构清晰**: 合理的章节划分

## 文档规范

### 标题规范
- 页面标题：1-3个词的短语
- 描述：一句话，5-10个词，不以"The"开头，避免重复标题
- 章节标题：简短，首字母大写，使用祈使语气

### 内容规范
- 文本块不超过2句话
- 章节之间用3个短横线分隔
- 避免在章节标题中重复页面标题的术语

### 代码规范
- JS/TS代码：移除尾随分号和不必要的逗号
- 提供完整可运行的示例
- 添加必要的注释
