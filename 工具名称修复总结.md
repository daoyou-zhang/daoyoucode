## 实际可用的搜索工具

根据 `backend/daoyoucode/agents/tools/registry.py`，实际注册的搜索工具有：

### 1. text_search（文本搜索）

**工具类**: `TextSearchTool`  
**用途**: 在文件中搜索文本（类似 ripgrep）  
**参数**:
- `query`: 搜索关键词
- `directory`: 搜索目录（默认 "."）
- `file_pattern`: 文件名模式（如 "*.py"）
- `case_sensitive`: 是否区分大小写
- `max_results`: 最大结果数

**使用场景**: 简单的文本搜索

### 2. regex_search（正则搜索）

**工具类**: `RegexSearchTool`  
**用途**: 使用正则表达式搜索  
**参数**:
- `pattern`: 正则表达式模式
- `directory`: 搜索目录（默认 "."）
- `file_pattern`: 文件名模式
- `case_sensitive`: 是否区分大小写
- `max_results`: 最大结果数

**使用场景**: 复杂的模式匹配

### 3. semantic_code_search（语义搜索）

**工具类**: `SemanticCodeSearchTool`  
**用途**: 按问题语义检索相关代码块  
**参数**:
- `query`: 自然语言查询
- `top_k`: 返回结果数量

**使用场景**: 理解功能、查找实现

## 工具选择建议

### 场景1：查找特定字符串
```
使用: text_search
示例: text_search(query="UserLogin", file_pattern="**/*.py")
```

### 场景2：复杂模式匹配
```
使用: regex_search
示例: regex_search(pattern="class \w+Agent", file_pattern="**/*.py")
```

### 场景3：理解功能实现
```
使用: semantic_code_search
示例: semantic_code_search(query="登录功能实现", top_k=8)
```

### 场景4：查找类/函数定义
```
使用: repo_map（最高效）
示例: repo_map(repo_path=".", max_depth=3)
```
