## 工作流中需要检查的工具名称

### 可能的错误

工作流中可能使用了不存在的工具名称，需要检查：

❌ **不存在的工具**：
- `grep_search` → 应该是 `text_search` 或 `regex_search`
- `batch_read` → 应该是 `batch_read_files`
- `batch_write` → 应该是 `batch_write_files`

✅ **正确的工具名称**：
- `text_search` - 文本搜索
- `regex_search` - 正则搜索
- `semantic_code_search` - 语义搜索
- `repo_map` - 代码地图
- `get_file_symbols` - 文件符号
- `lsp_diagnostics` - LSP 诊断
- `lsp_find_references` - LSP 查找引用
- `lsp_symbols` - LSP 符号
- `search_replace` - 搜索替换
- `intelligent_diff_edit` - 智能 Diff 编辑
- `write_file` - 写入文件
- `read_file` - 读取文件
- `batch_read_files` - 批量读取
- `batch_write_files` - 批量写入
- `git_diff` - Git 差异
