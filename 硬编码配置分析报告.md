# 硬编码配置分析报告

## 概述
本报告列出了代码中所有应该可配置但被硬编码的参数，按优先级和影响范围分类。

共发现 **12 个主要硬编码配置项**，分为：
- 🔴 高优先级: 4 项（影响核心功能）
- 🟡 中优先级: 4 项（影响性能和用户体验）
- 🟢 低优先级: 4 项（影响较小）

## 🔴 高优先级（影响核心功能）

### 1. 工具调用循环次数
**位置**: `backend/daoyoucode/agents/core/agent.py`
**当前值**: 函数参数 `max_iterations`（调用时传入，但没有全局默认配置）
**问题**: 
- 每次调用都需要手动指定，容易遗漏
- 不同 Skill 可能需要不同的迭代次数
- 测试代码中硬编码为 3 或 5

**影响**: 所有 Agent 的工具调用循环

**建议配置位置**:
```yaml
# config/agent_config.yaml
agent:
  max_tool_iterations: 10  # 默认最大工具调用次数
  max_tool_iterations_per_skill:  # 按 Skill 自定义
    code-assistant: 15
    chat-assistant: 5
```

**代码修改示例**:
```python
# 修改前
for iteration in range(max_iterations):
    ...

# 修改后
from .config_manager import get_config_manager
config = get_config_manager()
max_iterations = config.get('agent_config.agent.max_tool_iterations', 10)
for iteration in range(max_iterations):
    ...
```

### 2. LLM 请求超时时间
**位置**: `backend/daoyoucode/agents/llm/clients/unified.py:112, 192`
**当前值**: `timeout=1800.0` (30分钟)
**问题**:
- 硬编码在代码中，无法根据任务类型调整
- 同步和流式请求都使用相同超时
- 对于简单任务，30分钟太长；对于复杂任务可能不够

**影响**: 所有 LLM API 调用

**建议配置位置**:
```yaml
# config/llm_config.yaml
llm:
  timeout:
    default: 1800  # 默认30分钟
    simple_task: 300  # 简单任务5分钟
    complex_task: 3600  # 复杂任务1小时
    streaming: 1800  # 流式请求
```

**代码修改示例**:
```python
# 修改前
timeout=1800.0

# 修改后
from .config_manager import get_config_manager
config = get_config_manager()
timeout = config.get('llm_config.llm.timeout.default', 1800)
```

**影响代码**:
- `backend/daoyoucode/agents/llm/clients/unified.py:112` (同步请求)
- `backend/daoyoucode/agents/llm/clients/unified.py:192` (流式请求)

### 4. 缓存配置
**位置**: `backend/daoyoucode/agents/core/cache.py:306-308`
**当前值**: 
- `default_ttl=300` (5分钟)
- `max_size=1000`

**问题**:
- 不同类型的缓存应该有不同的 TTL
- 缓存大小应该根据系统内存调整

**影响**: 所有缓存功能的性能和内存使用

**建议配置位置**:
```yaml
# config/cache_config.yaml
cache:
  global:
    default_ttl: 300  # 5分钟
    max_size: 1000
  
  profile:  # 用户画像缓存
    ttl: 3600  # 1小时
    max_size: 500
  
  summary:  # 摘要缓存
    ttl: 1800  # 30分钟
    max_size: 200
  
  history:  # 历史缓存
    ttl: 600  # 10分钟
    max_size: 1000
  
  tool_result:  # 工具结果缓存
    ttl: 300  # 5分钟
    max_size: 500
```

**代码修改示例**:
```python
# 修改前
_global_cache = SimpleCache(
    default_ttl=300,
    max_size=1000
)

# 修改后
from .config_manager import get_config_manager
config = get_config_manager()
_global_cache = SimpleCache(
    default_ttl=config.get('cache_config.cache.global.default_ttl', 300),
    max_size=config.get('cache_config.cache.global.max_size', 1000)
)
```

## 🔧 配置管理器实现示例

```python
# backend/daoyoucode/agents/core/config_manager.py

from pathlib import Path
import yaml
from typing import Dict, Any, Optional

class ConfigManager:
    """统一配置管理器"""
    
    def __init__(self):
        self.configs = {}
        self._load_all_configs()
    
    def _load_all_configs(self):
        """加载所有配置文件"""
        config_files = [
            'agent_config.yaml',
            'llm_config.yaml',
            'cache_config.yaml',
            'lsp_config.yaml',
            'memory_config.yaml',
            'recovery_config.yaml',
            'repomap_config.yaml',
            'vector_config.yaml',
            'ui_config.yaml',
            'hooks_config.yaml',
            'paths_config.yaml',
            'middleware_config.yaml',
        ]
        
        for config_file in config_files:
            self._load_config(config_file)
    
    def _load_config(self, filename: str):
        """加载单个配置文件（支持多层覆盖）"""
        config_name = filename.replace('.yaml', '')
        
        # 1. 默认配置
        default_path = Path(__file__).parent.parent.parent / 'config' / 'defaults' / filename
        config = self._read_yaml(default_path) if default_path.exists() else {}
        
        # 2. 用户配置（覆盖默认）
        user_path = Path.home() / '.daoyoucode' / 'config' / filename
        if user_path.exists():
            user_config = self._read_yaml(user_path)
            config = self._deep_merge(config, user_config)
        
        # 3. 项目配置（覆盖用户）
        project_path = Path('.daoyoucode') / 'config' / filename
        if project_path.exists():
            project_config = self._read_yaml(project_path)
            config = self._deep_merge(config, project_config)
        
        # 4. 环境变量（覆盖项目）
        config = self._apply_env_vars(config, config_name)
        
        self.configs[config_name] = config
    
    def get(self, key: str, default: Any = None) -> Any:
        """获取配置值（支持点号路径）"""
        keys = key.split('.')
        config = self.configs
        
        for k in keys:
            if isinstance(config, dict):
                config = config.get(k)
                if config is None:
                    return default
            else:
                return default
        
        return config
    
    @staticmethod
    def _read_yaml(path: Path) -> Dict[str, Any]:
        """读取 YAML 文件"""
        with open(path, 'r', encoding='utf-8') as f:
            return yaml.safe_load(f) or {}
    
    @staticmethod
    def _deep_merge(base: Dict, override: Dict) -> Dict:
        """深度合并字典"""
        result = base.copy()
        for key, value in override.items():
            if key in result and isinstance(result[key], dict) and isinstance(value, dict):
                result[key] = ConfigManager._deep_merge(result[key], value)
            else:
                result[key] = value
        return result
    
    @staticmethod
    def _apply_env_vars(config: Dict, prefix: str) -> Dict:

        # 例如: DAOYOUCODE_AGENT_MAX_TOOL_ITERATIONS=15

# 全局单例
_config_manager: Optional[ConfigManager] = None

def get_config_manager() -> ConfigManager:
    """获取配置管理器单例"""
    global _config_manager
    if _config_manager is None:
        _config_manager = ConfigManager()
    return _config_manager
```
